Key Interactions

  • Toggle: Header button + Cmd+3 keyboard shortcut
  • Mode switch: "Branch" vs "Working" segmented control in sidebar header
  • Click file: Terminal area replaced with DiffView for that file; back button returns to terminal
  • Expand/collapse dirs: Click directory row to toggle children
  • Resize: Drag the ResizableDivider between terminal and sidebar
  • Status indicators: A = added (green), M = modified (yellow), D = deleted (red)
  • Default state: Collapsed — no sidebar visible, terminal fills full detail area
  • Width persistence: Sidebar width saved to @AppStorage, restored on next open
